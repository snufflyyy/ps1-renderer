# 🌊 SeaChess (*C* Chess)

 - Written in C99
 - SDL3 for window creation, creating the OpenGL context, input, and audio.
 - OpenGL 4.6 on Linux and Windows, and OpenGL 4.1 on MacOS.
 - Offline play using chess engines.
 - Online P2P play using TCP.
 - Custom UI library with animations.
 - JSON theme saving and loading.
